## Part 1 ✍️ 作文素材积累

### 素材三：名言金句 · 过渡句参考

> 过渡句的核心功能：**引出名言 + 连接论点**，让名言自然嵌入你的论证逻辑中。没有过渡句的名言引用就像硬塞进去的，阅卷老师一眼就能看出来。

#### 1. "苟利国家生死以，岂因祸福避趋之。" ——林则徐

**参考过渡句（示例）：**

> **①（爱国/担当主题）**
> 面对时代的考卷，真正的担当从来不是权衡利弊后的"聪明选择"，而是一种近乎本能的使命感——正如林则徐所言："苟利国家生死以，岂因祸福避趋之。"

> **②（个人选择主题）**
> 当个人得失与家国需要发生碰撞时，该如何抉择？林则徐用一句"苟利国家生死以，岂因祸福避趋之"给出了掷地有声的回答。

> **③（奉献/牺牲主题）**
> 奉献精神不是喊出来的，是在危难时刻站出来的决然——"苟利国家生死以，岂因祸福避趋之"，这份胸襟穿越百年，依然滚烫。

**得分要点：**
- ✅ 先点明主题（担当/选择/奉献），再引名言
- ✅ 名言不能"裸奔"，前后必须有分析性语言包裹
- ✅ 避免"正如某某说过"这种生硬过渡，用"用……给出了回答""道出了……"等更自然的表达

---

#### 2. "博观而约取，厚积而薄发。" ——苏轼

**参考过渡句（示例）：**

> **①（积累/坚持主题）**
> 所有看似"一夜爆红"的背后，都藏着日复一日的沉淀。苏轼说得好："博观而约取，厚积而薄发。"真正的成功，从来不是急功近利的结果，而是长期积累后的自然绽放。

> **②（学习/成长主题）**
> 在信息爆炸的时代，我们更需要"博观而约取"的智慧——广泛涉猎、审慎筛选，将知识的养分真正内化为自己的力量，才能在关键时刻"薄发"出令人惊艳的光芒。

> **③（创新/厚积薄发主题）**
> 创新不是凭空而来，它需要深厚的积累作为根基。正所谓"博观而约取，厚积而薄发"，没有千百次的失败和思考，就不会有灵光乍现的那一刻。

**得分要点：**
- ✅ 理解"约取"的含义：不是什么都取，而是精选精取
- ✅ 理解"薄发"的含义：不是不发力，而是积累到位后的精准输出
- ✅ 过渡时可以适当解释名言含义，再连接论点

---

#### 3. "知之者不如好之者，好之者不如乐之者。" ——孔子

**参考过渡句（示例）：**

> **①（热爱/兴趣主题）**
> 兴趣是最好的老师，热爱是最强的内驱力。孔子早就看透了这层道理："知之者不如好之者，好之者不如乐之者。"当你真正乐在其中时，学习便不再是负担，而是一种享受。

> **②（教育/成长主题）**
> 学习的最高境界不是"考了多少分"，而是发自内心地享受探索的过程。孔子将学习分为三重境界——知之、好之、乐之，而"乐之"才是我们应该追求的终点。

> **③（职业/人生选择主题）**
> 选择一条路，最重要的不是别人觉得好不好，而是你愿不愿意为之着迷。正如孔子所言："知之者不如好之者，好之者不如乐之者。"唯有热爱，才能让你在漫漫长路上不觉疲倦。

**得分要点：**
- ✅ 搞清楚"三重境界"的递进关系：知道→喜欢→乐在其中
- ✅ 过渡时点明这种递进，而不是简单引名言
- ✅ 可以结合当代语境（如内驱力、心流状态等）做现代化阐释，加分项

### 💡 微写作练习 · 范文参考

#### 选题A：谈谈你对"传承"的理解（示例）

> 传承，不是把灰掸干净供在橱窗里，而是让老手艺长出新的枝丫。当一群00后非遗传承人用短视频复原一把唐代琵琶时，他们做的不仅仅是"保护"，更是让传统文化以当代的方式被看见、被理解、被喜爱。传承的力量不在于"原封不动"，而在于让每一代人都能从中找到属于自己的共鸣。正如苏轼所言："博观而约取，厚积而薄发。"先广泛吸收，再精准传递——这才是传承最生动的模样。（167字）

**亮点分析：**
- 开头用比喻破题，观点鲜明
- 引用素材二（非遗传承人）+ 素材三（苏轼名言），双重素材联动
- 结尾用名言收束，首尾呼应
- 字数控制在150-200区间

---

#### 选题B："选择"之我见（示例）

> 什么是真正的选择？不是站在十字路口的犹疑不决，而是在万千诱惑面前的笃定从容。钱学森面对美国当局的威逼利诱，用五年等待和一句"我的归宿在中国"给出了自己的答案。选择之所以艰难，往往不是因为选项不够好，而是因为我们内心缺少一个足够坚定的锚点。当林则徐说出"苟利国家生死以，岂因祸福避趋之"时，他告诉我们：最好的选择，不在于计算得失，而在于问心无愧。（171字）

**亮点分析：**
- 开头设问+自答，直接亮明观点
- 引用素材一（钱学森）+ 素材三（林则徐），双素材叠加
- "锚点""问心无愧"等用词有力度
- 首尾呼应，结构完整

---

#### 选题C：热爱的力量（示例）

> 热爱是什么？它是深夜画室里不肯熄灭的灯，是练习室里反复摔倒又站起的身影，也是宋濂在砚冰坚、手指不可屈伸的寒冬里依然不愿停笔的执念。孔子说："知之者不如好之者，好之者不如乐之者。"当你真正热爱一件事时，困难不再是阻碍，而成了通往目标的阶梯。那些非遗传承的年轻人，他们不是在"坚持"，而是在"享受"——因为热爱，所以甘之如饴。（163字）

**亮点分析：**
- 排比开头，节奏感强
- 巧妙把 Part 2 的宋濂素材也用上了（加分！）
- 引用孔子名言，扣住"乐之"的境界
- 结尾"甘之如饴"收得漂亮

## Part 2 📖 文言文翻译练习 · 参考答案

### 选文：《送东阳马生序》（节选）

---

#### 第1句：余幼时即嗜学。

**字词解析：**

| 字词 | 释义 |
|------|------|
| 余 | 我（第一人称代词） |
| 幼时 | 小时候 |
| 即 | 就（副词，表示早、已经） |
| 嗜 | **特别喜爱，酷爱**（注意：不只是"喜欢"，程度更深，近乎痴迷） |
| 学 | 学习 |

**参考翻译：**
> 我小时候就特别喜爱学习。

**易错提醒：**
- ❌ "我喜欢学习"——程度不够，"嗜"是"极度喜爱"的意思
- ❌ 漏掉"即"字——"即"强调"就/已经"，表示从小就有这种倾向
- ✅ 翻译要体现"嗜"的程度之深

**得分点：** "嗜"译为"特别喜爱/酷爱"得1分，全句通顺得1分。

---

#### 第2句：家贫，无从致书以观，每假借于藏书之家，手自笔录，计日以还。

**字词解析：**

| 字词 | 释义 |
|------|------|
| 无从 | 没有办法 |
| 致 | **买到，得到**（注意：不是"致信"的"致"，这里是"获得"的意思） |
| 以 | 连词，表目的，"用来" |
| 观 | 阅读 |
| 每 | 每次，往往 |
| 假借 | **借**（"假"=借，"借"=借，同义复合词） |
| 于 | **介词，"向"**（引出对象，"向藏书的人家借"） |
| 手自 | 亲手 |
| 笔录 | 抄写（"笔"名词作状语=用笔） |
| 计日 | 按约定的日期 |
| 以 | 连词，表修饰或顺承 |
| 还 | 归还 |

**参考翻译：**
> （因为）家境贫寒，没有办法买书来看，就经常向藏书的人家借，亲手抄写，按约定的日期归还。

**句式要点：**
- **"假借于藏书之家"** = 介宾短语后置（状语后置句），正常语序为"每（于藏书之家）假借"。翻译时要把"向藏书的人家"放到动词"借"前面。
- **"手自笔录"** = "笔"名词作状语，"用笔抄写"，不是"亲手笔记"。

**易错提醒：**
- ❌ "致书"翻译成"写信"——完全错误！这里是"得到/买到书"
- ❌ "假借"翻译成"虚假借用"——"假"在这里是"借"的意思
- ❌ "于"翻译成"在"——这里是"向"，表示动作对象
- ❌ 忽略"笔"的名词作状语用法

**得分点：** "致"译对1分，"假借"译对1分，"于"的用法1分，全句通顺1分。

---

#### 第3句：天大寒，砚冰坚，手指不可屈伸，弗之怠。

**字词解析：**

| 字词 | 释义 |
|------|------|
| 大寒 | 非常寒冷 |
| 砚冰坚 | 砚台里的墨水结成了坚硬的冰 |
| 屈伸 | 弯曲和伸展 |
| 弗 | 不（否定副词） |
| 之 | 代词，指代"抄书这件事" |
| 怠 | 松懈，怠慢，放松 |

**特殊句式：**
> **"弗之怠" = 宾语前置句**
> 正常语序为"弗怠之"（不放松抄书这件事）。
> 否定句中代词作宾语，宾语前置。这是文言文常见句式。

**参考翻译：**
> 天气非常寒冷，砚台里的墨水结成了坚硬的冰，手指不能弯曲伸展，（我也）不放松抄书这件事。

**易错提醒：**
- ❌ "弗之怠"翻译成"没有怠慢"——没说清楚"怠"的对象是什么
- ❌ 没识别出宾语前置句式——考试中要求指出句式类型
- ❌ "冰坚"翻译成"冰很坚硬"——要补出"砚台里的墨水"这个主语

**得分点：** 指出宾语前置句式1分，"弗之怠"译对1分，全句通顺1分。

---

#### 第4句：录毕，走送之，不敢稍逾约。以是人多以书假余，余因得遍观群书。

**字词解析：**

| 字词 | 释义 |
|------|------|
| 毕 | 完毕，结束 |
| 走 | **跑**（注意：古汉语中"走"="跑"，不是现代的"步行"！） |
| 之 | 代词，指代借来的书 |
| 稍 | 稍微，一点儿 |
| 逾约 | 超过约定的期限 |
| 以是 | **因此，因为这个原因** |
| 以 | 介词，"把" |
| 假 | **借给**（注意：和第2句的"假借"呼应，这里是单用"假"） |
| 余 | 我 |
| 因 | 于是，因此 |
| 得 | 能够 |
| 遍观 | 广泛阅读，全部看完 |
| 群书 | 各种书籍 |

**参考翻译：**
> 抄写完毕，（我）跑着把书送还给人家，不敢稍微超过约定的期限。因此人们大多把书借给我，我于是能够广泛阅读各种书籍。

**古今异义重点：**
- **"走"**：古义=跑，今义=步行。这是高频考点，务必记住！
- 类似的还有："去"古义=离开，今义=前往；"妻子"古义=妻子和儿女，今义=妻子。

**句式要点：**
- **"以书假余"** = 双宾语结构，"把书借给我"，"余"是间接宾语。
- **"以是"** = 固定结构，"因此"，翻译时放在句首。

**易错提醒：**
- ❌ "走"翻译成"走"（步行）——古文中"走"是"跑"！这是最常考的古今异义词之一
- ❌ "以是"翻译成"以为"——完全不是一回事，"以是"="因此"
- ❌ "假余"翻译成"假借给我"——"假"本身就有"借给"的意思，不需要加"借"

**得分点：** "走"译为"跑"1分，"以是"译对1分，"假"译为"借给"1分，全句通顺1分。

## 🎯 翻译方法总结（考试通用）

### 四步翻译法：

1. **逐字对应**：先给每个字找到对应的现代汉语释义
2. **补充省略**：古文省略主语、宾语很常见，翻译时要补出来（用括号标注）
3. **调整语序**：宾语前置、状语后置、定语后置等特殊句式，要调整为现代语序
4. **润色通顺**：确保译文读起来是通顺的现代汉语，不留文言腔

### 本课重点实词回顾：

| 字词 | 本课释义 | 其他常见释义 | 记忆口诀 |
|------|----------|------------|---------|
| 嗜 | 特别喜爱 | 嗜好、嗜酒 | "嗜"比"好"更深一层 |
| 致 | 得到、买到 | 导致、精致 | "致书"=弄到书 |
| 假 | 借 | 假如、真假 | "假借"不是"虚假" |
| 走 | 跑 | （今义=步行） | "奔走相告"里的"走"=跑 |
| 逾 | 超过 | 逾期、逾越 | "逾约"=超过约定 |
| 弗 | 不 | 弗去=不去 | "弗"="不" |

### 本课重点虚词回顾：

| 虚词 | 本课用法 | 举例 |
|------|----------|------|
| 以 | ① 连词，表目的（用来） ② 介词（把） | "无从致书以观" / "以书假余" |
| 于 | 介词，向 | "假借于藏书之家" |
| 之 | ① 代词（指书） ② 宾语前置的标志 | "走送之" / "弗之怠" |
