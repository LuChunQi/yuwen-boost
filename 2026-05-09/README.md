## 📖 第一部分：素材阅读（5分钟）

先读下面这段素材，然后完成写作任务。

### 素材一：汪曾祺《人间草木》节选

> 如果我还在家乡，我大概还会养花。我爱花，但不会养花。我养过很多种花，都没有养好。倒是随手种的一些东西，反而长得茂盛。有一次在院子里吃西瓜，把瓜子随便吐在地上，后来居然长出几棵瓜苗，开了花，还结了一个小西瓜。这就是所谓的"无心插柳柳成荫"吧。
>
> 人做事也是这样，太刻意了反而做不好。放松一点，随意一点，说不定会有意想不到的收获。

### 素材二：热点话题

**2026年五一假期旅游数据**：今年五一假期全国国内旅游出游合计3.2亿人次，同比增长8.4%。然而，社交媒体上"反向旅游"（选择小众目的地避开人潮）成为年轻人的热门选择。有人追求"打卡热门景点"，有人偏爱"安静待在一个地方"，你怎么看？

## ✍️ 第二部分：写作练习（25分钟）

### 任务：从以下两个题目中**任选一个**，写一篇不少于400字的文章

---

### 🔵 题目一：微作文 + 拓展

**题目：「无意之间」**

生活中很多时候，最好的结果不是刻意追求来的，而是在无意之间发生的。正如汪曾祺随手吐的西瓜子，反而长出了果实。

**要求：**
1. 先写一个 **80-120字的微语段**，描写一次"无意之间"的经历或发现（可以是生活中的小事）
2. 在微语段基础上，**扩展为一篇400字以上的文章**，题目自拟（或用「无意之间」）
3. 文体不限（记叙文、散文、议论文均可）
4. 要求有 **具体细节**——不要全是抽象议论

**💡 写作提示：**
- 可以写无意间听到的一句话、无意间发现的一个地方、无意间交到的一个朋友……
- 关键是写出 **"无意"到"有意义"的转变过程**
- 结尾可以稍微升华，但别空喊口号

---

### 🟢 题目二：议论文片段

**题目：「打卡」与「在场」**

有人说旅行一定要去热门景点打卡，才不算白来；也有人说，真正的旅行是安静地感受一个地方，拍不拍照不重要。

**要求：**
1. 写一篇 **400字以上** 的议论文
2. 明确提出你的观点
3. 至少使用 **一个具体事例** 作为论据（可以来自素材、自己的经历、或你知道的故事）
4. 有开头、有论证、有结尾，结构完整

**💡 写作提示：**
- "打卡"不只是旅游的事——学习、社交、看展览……到处都有"打卡心态"
- 可以从材料出发，但要联系更广的范围
- 反面论证也很好用："如果所有人都去打卡，那真正被忽略的是什么？"

## 🔍 第三部分：写完自查清单

写完后，对照这个清单检查一遍（**不求全做到，但至少关注3条**）：

| 序号 | 检查项 | ✅做到了？ |
|------|--------|-----------|
| 1 | 开头是否吸引人？（不是"随着社会的发展……"这种） | |
| 2 | 有没有至少一个 **具体的细节描写**？（不是空话） | |
| 3 | 用词有没有重复？能不能换个说法？ | |
| 4 | 结尾有没有"硬凑"的升华？能不能写得更自然？ | |
| 5 | 标点符号用对了吗？（尤其是引号、破折号、省略号） | |
| 6 | 有没有错别字或病句？大声读一遍试试 | |

## 🌟 加分挑战（选做）

如果你觉得400字还不够过瘾，试试这个：

> 用 **一句话**（不超过30字）概括你今天文章的核心意思。
>
> 比如：「真正的旅行不是去了哪里，而是感受到了什么。」

这句话就是你的"文眼"。学会提炼文眼，考试作文立意会更清晰。
