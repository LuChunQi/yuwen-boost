**春天里的发现**

①每年春天，我都会回到乡下老家住几天。不为别的，就想看看那些在冬天里消失的东西，是怎样一件一件回来的。

②最先回来的是声音。冬天的村庄太安静了，安静得让人不自在。风是干燥的，吹在脸上像砂纸打磨。鸟也不怎么叫，偶尔有一只麻雀在屋檐下瑟缩着，发出微弱的吱吱声，像是怕惊动了什么。但立春一过，情况就不同了。先是一种很细很细的嗡嗡声，不知道是从哪里来的，也说不清是什么在响，但你就是感觉空气在震动，像有人在你耳边轻轻哼了一个音。然后这嗡嗡声慢慢扩散，慢慢清晰，有一天你突然意识到——那是冰在裂开的声音。河面的冰、地上的冰、树皮里的冰，都在用一种极低极慢的速度碎裂，发出一种极细极温柔的声音。

③声音回来之后，颜色就跟着回来了。最先回来的颜色不是绿色，而是黄色。是的，是迎春花。它几乎是一夜之间冒出来的，前一晚还什么都没有，第二天早上推开门，墙根下就多了一丛明晃晃的黄。那黄色太亮了，亮得不像是真的，像是有人趁你睡觉的时候用颜料刷上去的。迎春花之后是杏花，粉白的，一团一团挂在枝头，远远看去像低低的云。然后是桃花，颜色比杏花深一些，也艳一些，站在树下仰头看，满眼都是粉红色，恍惚间觉得自己也变成了一瓣花。

④但春天真正让我着迷的，不是声音和颜色，而是气味。泥土的气味。准确地说，是解冻之后潮湿的泥土散发出来的那种气味。那是一种很难形容的味道，有点腥，有点甜，有点像你小时候把脸埋在妈妈刚洗好的头发里的感觉。这气味只有春天才有，也只有回到泥土跟前才能闻到。城市里的春天也有气味，但那不一样。城市里你闻到的是沥青被晒软的味道、空调外机排出的温热气体的味道、奶茶店飘出来的甜腻味道。那不是春天的气味，那是城市在模仿春天。

⑤我很害怕有一天我再也闻不到泥土的气味了。不是因为我不再回乡下，而是因为乡下也变了。每次回去，村里的人都会少几个。老人走了，年轻人不回来，孩子们在镇上上学。我小时候认识的那片麦田，去年已经被推平了，说要建一个什么加工厂。那块地里曾经有我最喜欢的一条田埂，两边种着蚕豆，春天开紫黑色的花，风一吹就哗啦啦响。现在那里什么都没有了，只有一块平坦的、光秃秃的黄土。

⑥也许有人会说，你怀念的不是春天，是童年。也许吧。但我宁愿相信，春天的魔力在于它每年都让你产生一种错觉：一切都可以重新开始。冰可以化，花可以开，泥土可以重新散发气味。哪怕你已经见过三四十个春天了，第三个十一个和第一个之间，隔着的是一整个漫长的人生。但当你再次闻到那股泥土的气味时，你会觉得，一切确实可以重新开始。

## 💡 做题提示

- **现代文阅读**：注意从"写了什么→怎么写的→为什么这样写"的思路去分析，不要空泛地说"生动形象"。
- **病句题**：重点关注两面失衡（能否/是否）、成分残缺（通过…使…）、否定失当（防止…不…）这三大高频错因。
- **成语题**：注意区分"不刊之论"（正确的，不可删改的）和望文生义的用法。
