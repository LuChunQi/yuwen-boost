## 一、名言素材积累（先读后用）

请认真阅读以下三条素材，它们都可以用在今天的作文中：

### 素材 1
> "千里之堤，溃于蚁穴。" ——《韩非子·喻老》

**解读**：小事不注意，大问题就会找上门。这句话适合写"细节""积累""防微杜渐"等主题。

### 素材 2
> "不积跬步，无以至千里；不积小流，无以成江海。" ——荀子《劝学》

**解读**：所有伟大的成就都是一步步积累来的。适合写"坚持""积累""成长"等主题。

### 素材 3
> "天下难事，必作于易；天下大事，必作于细。" ——老子《道德经》

**解读**：再难的事也要从简单处着手，再大的事也要从细节做起。适合写"脚踏实地""从小事做起"等主题。

## 二、小练笔：用素材写一段话（15分钟）

**要求**：从上面的三条素材中**至少选一条**，写一段 200-300 字的议论片段。

**题目**：《从一件小事说起》

**提示**：
- 你可以写一件真实的小事（早起、背单词、练字……），然后用素材来升华
- 也可以反向写——因为忽视小事而吃亏的经历
- 关键是：**叙述 + 议论结合**，不能只讲故事不分析

**结构建议**：
1. 开头：用一句素材引出观点（1-2句）
2. 中间：举一个具体的小事例（3-4句）
3. 结尾：回到素材，点题升华（1-2句）

## 三、作文实战：完成一篇完整作文（30分钟）

**题目**：**《微小》**

**要求**：
- 文体不限（推荐散文或议论文）
- 不少于 600 字
- 必须使用至少一条名言素材（可以用上面提供的，也可以用自己的积累）

**写作提示**：

### 如果写议论文：
- **破题方向**：微小的力量/微小中的伟大/不可忽视的微小
- **可用分论点**：
  - 微小的积累能产生巨大的变化（跬步→千里）
  - 微小的细节决定成败（蚁穴→溃堤）
  - 微小的善举能温暖世界（蝴蝶效应）
- **结尾建议**：回应"微小"这个题目，点明你的核心观点

### 如果写散文：
- **破题方向**：一个微小的人/一件微小的事/一个微小的瞬间
- **叙事线索**：从一个具体的"微小"切入，慢慢展开
- **情感节奏**：平淡开头 → 细节铺陈 → 情感升华
- **结尾建议**：用一句感悟或素材收束，余韵悠长
