## 一、名言素材积累 · 解析

今天的练习是作文专项，没有"标准答案"，但每条素材有明确的使用要点和常见误区。

### 素材 1："千里之堤，溃于蚁穴"

| 维度 | 要点 |
|------|------|
| **适用主题** | 细节、防微杜渐、危机意识、质量把控 |
| **正确用法** | 用在"忽视小事→酿成大祸"的逻辑链上，强调**因果关系** |
| **常见误用** | ❌ 只引用不分析，变成"贴标签"——"正如千里之堤溃于蚁穴，所以我们要注意细节。"（太浅） |
| **高阶用法** | ✅ 先叙述一个因忽视细节而失败的案例，再引用此句收束："一座大坝的崩塌，往往始于一条不起眼的裂缝。正如古人所言，千里之堤，溃于蚁穴。那些被我们忽略的'蚁穴'，恰恰是命运的伏笔。" |

### 素材 2："不积跬步，无以至千里；不积小流，无以成江海"

| 维度 | 要点 |
|------|------|
| **适用主题** | 坚持、积累、成长、量变到质变 |
| **正确用法** | 举例说明"每天做一点小事→最终达成大目标"的过程 |
| **常见误用** | ❌ 滥用为万能开头，和后文内容脱节 |
| **高阶用法** | ✅ 拆开使用，把"跬步"和"小流"拆成两个分论点的论据，形成排比结构 |

### 素材 3："天下难事，必作于易；天下大事，必作于细"

| 维度 | 要点 |
|------|------|
| **适用主题** | 从小事做起、脚踏实地、方法论 |
| **正确用法** | 强调"做"的路径——从简单处入手，从细节处突破 |
| **常见误用** | ❌ 和素材2混淆，把"方法论"写成"坚持论" |
| **高阶用法** | ✅ 结合个人经历，写"面对一个看似不可能的任务，我从最小的一步开始……" |

## 二、小练笔：《从一件小事说起》· 范文参考与点评

### 🌟 范文参考（约 250 字）

> 荀子说"不积跬步，无以至千里"，这话听起来老生常谈，但真正做到的人却不多。去年暑假，我决定每天早起跑步，第一天信心满满地绕操场跑了三圈，第二天腿酸得起不来，第三天就放弃了。后来我换了个方式——每天只跑一圈，不给自己压力。一圈变成了两圈，两圈变成了三圈，一个月后我居然能轻松跑完两公里。回头看，那些看似微不足道的一圈一圈，就是"跬步"。它们不起眼，却真真切切地把我带到了一个之前到不了的地方。原来，"小事"的意义不在于它本身有多重要，而在于它是通往"大事"的第一级台阶。

### 📊 点评分点

| 评分维度 | 满分 | 范文得分 | 说明 |
|----------|------|----------|------|
| 素材运用 | 4分 | 3.5分 | 引用了素材2，开头引出观点，结尾呼应，但分析可以再深一层 |
| 叙议结合 | 4分 | 3.5分 | 有具体事例（跑步），也有分析（"小事"的意义），结合较自然 |
| 结构完整 | 3分 | 3分 | 开头引题→中间叙事→结尾升华，结构清晰 |
| 语言表达 | 3分 | 2.5分 | 流畅自然，但个别句子可以更精练 |
| **总分** | **14分** | **12.5分** | 良好水平 |

### ⚠️ 常见问题提醒

1. **只叙事不议论**：写了很多故事细节，但忘了分析"为什么小事很重要"——这是最大的扣分项
2. **素材和内容脱节**：开头引了一句名言，后面写的事例跟名言毫无关系，阅卷老师一眼就能看出来
3. **议论太空洞**："小事很重要，我们一定要注意小事。"——说了等于没说，要有具体的逻辑推演

## 三、作文实战：《微小》· 范文参考与点评

### 🌟 议论文范文（约 700 字）

# 微小

老子言："天下难事，必作于易；天下大事，必作于细。"在浩瀚的宇宙面前，人是微小的；在漫长的历史面前，一个人的一生是微小的。然而，正是无数个"微小"的叠加，构成了世界的壮阔。

微小是一种力量。科学家研究发现，一只蜜蜂的翅膀每秒振动约两百次，产生微乎其微的气流。但当千万只蜜蜂同时振翅，那股气流足以改变一片花田的传粉格局。这就是"微小"的乘数效应——它不是静止的、被动的，而是一种蓄势待发的力量。每个人的日常努力，每一次翻书、每一道解题、每一页笔记，都是翅膀的一次振动。单独看，微不足道；汇聚起来，足以翻山越岭。

微小是一种态度。日本清洁工新津春子，在北京机场做了二十三年保洁工作。她不是什么大人物，没有惊天动地的故事，但她把每一块瓷砖都擦到反光，把每一个角落都清理到无可挑剔。人们称赞她"让机场发光"。她的"微小"不是无奈的妥协，而是一种主动的选择——把小事做到极致，本身就是一种伟大。反观那些总想着"做大事"却连房间都不整理的人，他们的"大"往往只是空谈。

微小也是一种智慧。围棋中，高手之间的差距往往只在半目之间。那看似微不足道的一手棋，可能决定整盘棋的胜负。人生亦如此——考试中一个粗心的计算错误，可能让总分差一个等级；面试时一个不经意的细节，可能影响整场评价。真正聪明的人，不是看不到微小，而是深知微小的分量，所以不敢轻忽。

当然，强调微小不是叫人只盯着脚下，不抬头看路。微小的意义，恰恰在于它是通往远方的第一步。千里之行，始于足下——那"足下"的一步，是最微小的，也是最关键的。

我们不必害怕自己的渺小，也不必轻视日常的琐碎。因为每一个"微小"的背后，都藏着一个更大的世界。

### 🌟 散文范文（约 650 字）

# 微小

外婆家的厨房里有一扇很小的窗，小到只能看见邻居院里的一棵桂花树。

小时候我不喜欢那扇窗。透过它，世界被裁成窄窄的一条——灰白的墙，一棵不太高的树，和偶尔掠过的鸟影。我把脸贴在窗玻璃上往外挤，想看到更多，但窗就那么大，挤破了头也看不见整片天空。

后来我到城里上学，教室的窗很大，一眼望出去是整条马路，车水马龙，热热闹闹。我以为这就是"开阔"，以为窗户越大，世界就越大。

直到初二的冬天。外婆住院了，我回去看她。病房的窗也很大，对面是另一栋住院楼，灰蒙蒙的，什么也看不见。外婆倒是笑眯眯的，说："病房的窗再大，也不如我厨房那扇小窗，至少还能看见桂花开了没有。"

我突然有点难过。

那扇小窗确实只能看见一棵树。但那棵树春天抽新芽，夏天撑开一片绿荫，秋天落满金色的小花，冬天光秃秃地站在风里也不倒下。外婆在厨房里做饭的时候，边炒菜边看那棵树，看了几十年。那棵树从一棵小苗长成比屋檐还高的大树，外婆的头发也从黑变白。

窗很小，但一棵树的一生都在里面。这算不算一种"微小"里的完整？

我想起小时候趴在窗台上看桂花，阳光从树叶缝隙里漏下来，落在灶台上，细碎得像金粉。外婆不让我摘桂花，说"让它自己落，落的才是香的"。那时候不懂，现在想想，好像懂了一点点——有些东西，因为小、因为近、因为一直在身边，反而被我们忽略了。可等你回头再看，才发现那些"微小"的瞬间，才是真正撑起记忆的东西。

外婆出院后，我又去看了那扇窗。桂花早就落完了，枝头光秃秃的。但我知道，明年秋天，它还会开。

窗还是那么小，但世界没有变小。

## 四、自检清单 · 核对指南

| 检查项 | 如何自查 | 常见问题 |
|--------|----------|----------|
| 字数≥600字 | 数一下行数，每行约25-30字 | 写到500字就停了 → 说明分论点不够展开 |
| 使用了名言素材 | 搜索文中是否有引用 | 引了但没分析 → 等于没用 |
| 标题"微小"得到呼应 | 文中至少出现2次以上与"微小"相关的表述 | 只在标题写了"微小"，正文完全不提 |
| 无明显错别字 | 重点检查"的/地/得""已/以""在/再" | 这三个是高中生高频出错点 |
| 无流水句 | 检查是否连续3个以上句号都是简单陈述 | 用问号、感叹号、破折号、分号打破节奏 |
| 结尾有力 | 读最后一段，问自己"如果只看结尾，能感受到力量吗？" | "让我们一起努力吧"→ 空洞；"窗还是那么小，但世界没有变小"→ 有力 |

## 📌 今日学习要点总结

1. **素材不是装饰品**：引用名言后必须有分析，把名言和你的论点焊接在一起
2. **小事要写出"大"来**：从具体到抽象，从个别到一般，这是议论文的基本功
3. **议论文的三段式分论点**：每个分论点从不同角度解读题目，避免重复
4. **散文的"物—情—理"递进**：从一个具体的物/场景出发，写出情感，最后提炼感悟
5. **自检不是走过场**：每次写完都认真过一遍清单，养成习惯，考场上能省5-10分
