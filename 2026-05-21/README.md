## 一、文言文阅读

### 《史记·淮阴侯列传》（节选）

信数与萧何语，何奇之。至南郑，诸将行道亡者数十人。信度何等已数言上，上不我用，即亡。何闻信亡，不及以闻，自追之。人有言上曰："丞相何亡。"上大怒，如失左右手。

居一二日，何来谒上，上且怒且喜，骂何曰："若亡，何也？"何曰："臣不敢亡也，臣追亡者。"上曰："若所追者谁？"何曰："韩信也。"上复骂曰："诸将亡者以十数，公无所追；追信，诈也。"何曰："诸将易得耳。至如信者，国士无双。王必欲长王汉中，无所事信；必欲争天下，非信无所与计事者。顾王策安所决耳。"

### 📝 题目

**1.** 下列句中加点词解释**不正确**的一项是：（   ）

A. 信**数**与萧何语（多次）  
B. **度**何等已数言上（估计、揣测）  
C. 不及以**闻**（听到）  
D. **顾**王策安所决耳（只是、不过）

**2.** 下列各组句子中，加点词意义和用法**相同**的一组是：（   ）

A. ①何**奇**之 / ②奇闻异事  
B. ①上**不我用** / ②秋毫不敢有所犯  
C. ①**如**失左右手 / ②至如信者  
D. ①**若**亡，何也 / ②若所追者谁

**3.** 把下列句子翻译成现代汉语：

（1）"诸将亡者以十数，公无所追；追信，诈也。"

（2）"至如信者，国士无双。王必欲长王汉中，无所事信；必欲争天下，非信无所与计事者。"

**4.** 萧何为什么要追回韩信？从这段文字中可以看出萧何怎样的性格特点？请简要分析。

## 三、文言小翻译（每日一练）

将以下短文翻译成现代汉语：

> 秦孝公据崤函之固，拥雍州之地，君臣固守以窥周室，有席卷天下、包举宇内、囊括四海之意，并吞八荒之心。
> —（《过秦论》贾谊）

**提示：**
- 注意"据""拥""窥"三个动词的准确翻译
- "席卷""包举""囊括"都是比喻用法，想想怎么翻译既准确又不失气势
- "八荒"指什么？
