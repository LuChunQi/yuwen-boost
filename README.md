# 📖 语文 Boost — 高中语文三年提升计划

> 每天30分钟，从及格线到125+

## 这是什么？

一份系统的高中语文提升课程，面向高中生，每天推送针对性练习。

- **时间跨度**：高一 → 高三（2026.5 - 2029.6）
- **每日投入**：约30分钟
- **目标**：从及格线附近提升到高考125+

## 课程安排

- **月度主题**：文言文启蒙 — 从课本出发，每天一篇短文

## 训练理念

1. **阅读为王**：大量阅读文言文和现代文，语感比刷题重要
2. **翻译驱动**：文言文翻译是核心能力，每天坚持练
3. **循序渐进**：难度逐步提升，不搞题海战术
4. **应试有度**：病句/成语等应试题保持手感，不喧宾夺主
5. **反馈闭环**：作文批改 → 点评 → 改进，形成正循环
