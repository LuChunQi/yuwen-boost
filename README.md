# 📖 语文 Boost — 高中语文三年提升计划

> 每天30分钟，从及格线到125+

## 这是什么？

一份系统的高中语文提升课程，面向高中生，每天推送针对性练习。

- **时间跨度**：高一 → 高三（2026.5 - 2029.6）
- **每日投入**：约30分钟
- **目标**：从及格线附近提升到高考125+

## 课程安排

### 周一：文言文阅读（15min）+ 古诗词（15min）
### 周二：文言文阅读（15min）+ 基础知识（15min）
### 周三：现代文阅读训练（30min）
### 周四：文言文阅读（15min）+ 古诗词（15min）
### 周五：作文素材积累（15min）+ 基础知识（15min）
### 周六：作文练习（30min）
### 周日：复习本周内容 + 错题整理

## 目录结构

```
2026-05-05/           # 每天一个目录（日期命名）

essays/               # 作文专项
├── week01-topic.md   # 题目 + 写作指导
├── week01-review.md  # 批改点评
└── materials.md      # 素材库

notes/                # 知识点总结
├── 文言文实词表.md
├── 古诗词意象手册.md
├── 答题模板.md
└── 成语积累.md

progress/             # 学习记录
└── monthly-report.md
```
